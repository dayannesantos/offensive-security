# Offensive Security

Repository focused on offensive security research, adversary simulation, lab development, tradecraft studies and security tooling.

The goal of this repository is to organize notes, experiments, detection-oriented offensive research and technical material related to real-world attack techniques and offensive operations.

---

## Structure

```text
offensive-security/
├── active-directory/
├── web-security/
├── cloud-security/
├── exploit-development/
├── labs/
├── tooling/
└── README.md
```

*   ACL abuse
*   BloodHound

#### Web Security

Studies and practical material involving:

*   XSS

*   Access control issues

#### Cloud Security

Offensive research involving:

*   AWS

#### Exploit Development

Content related to:

*   Binary exploitation

#### Labs

*   Attack simulation

#### Tooling

Security scripts, automations and operational utilities.
